What is it?

The 7-1-7 Tracking Tool is a web-based dashboard that allows ZNPHI to record, track, and analyse outbreak response timeliness data in line with the WHO 7-1-7 framework.

It digitises the existing WHO 7-1-7 Excel data consolidation spreadsheet into a live, multi-user system accessible on any device.

How to access
Admin loginFull dashboard access: overview, charts, events, bottlenecks, actions, import
User loginData entry form only: enter and edit outbreak events
DeviceAny browser on desktop, laptop, or mobile
OfflineData entry works without internet and syncs when connectivity returns
What it does
  • Captures full outbreak event details including emergence, detection, notification, all seven early response actions (RA1-RA7), bottlenecks, enablers, and EAR presentation date
  • Auto-calculates 7-1-7 timeliness intervals and colour-codes results against targets
  • Displays a live map of Zambia with province-level event density
  • Tracks and ranks bottlenecks across detection, notification, and response stages
  • Manages remedial actions with status tracking
  • Imports data from the WHO 7-1-7 Excel consolidation spreadsheet
  • Flags duplicate events and date sequence errors before data is saved
  • Syncs all data to a secure server so every authorised user sees the same information
  • Exports all data as JSON or CSV for further analysis
Data entry guide
Enter / Edit Event tab
Fill in the form from top to bottom. Fields mirror the Excel Sheet 1 exactly. The live 7-1-7 calculation panel appears automatically as you enter dates, showing whether each interval meets the target before you save.
Import data tab
Step 1: Connect to the server and test the connection. Step 2: Sync any locally saved events to the server. Step 3: Upload your Excel file to import existing data. Events with date errors or duplicates are automatically skipped and listed.
Validation rules
Detection on or after emergence. Notification on or after detection. All RA dates on or after notification. End date after emergence.
